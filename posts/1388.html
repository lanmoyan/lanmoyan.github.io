利用Hexo与Github搭建博客

发表于2022-10-02|更新于2022-10-09
|字数总计:154|阅读时长:1分钟|阅读量:

一、前言

Hexo 是一个快速、简洁且高效的博客框架。https://hexo.io/zh-cn/

Hexo 在使用 Markdown（或其他渲染引擎）解析文章，在几秒内即可利用靓丽的主题生成静态网页。

二、安装相关软件

1.安装 node.js 软件 官网链接:https://nodejs.org/en/download/

2.安装 Git 软件 官网链接:https://git-scm.com/downloads

3.安装 vscode 软件 官网链接:https://code.visualstudio.com/

4.Win + R 打开运行框，输入 cmd 后回车，输入以下命令，如出现版本号即视为安装成功

git —version

node -v

npm -v
